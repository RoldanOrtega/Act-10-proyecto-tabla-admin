## Prompt
"Hola, quiero crear una App en Flutter para administrar libros. La App se llama 'Lecturas'. Necesito que el diseño sea muy específico, oscuro y moderno. Sigue estas instrucciones paso a paso de forma simple:

1. Los Colores (Estética Cyber-Pink):

Fondo de pantalla: Todo debe ser NEGRO absoluto (#000000).

Cajitas y Campos: Los cuadros donde se escribe y los recuadros de fondo deben ser ROSA CLARO (#FFC0CB).

Botones Mágicos (Hover): Cuando pase el mouse sobre un botón, este debe cambiar de azul oscuro a ROSA FUERTE (#FF69B4).

Icono Maestro: El logo de la app será siempre un Libro Abierto (Icons.menu_book) en color ROSA FUERTE.

2. El Inicio de Sesión (Seguridad):

Solo existe un administrador con estos datos fijos:

Correo: lecturas@gmail.com

Contraseña: lecturas123

Si alguien se equivoca, muestra un aviso con una calavera que diga '¡ACCESO DENEGADO!'.

3. El Cerebro (Firebase):

Conecta la app a Firebase Firestore.

La colección se llama lecturas.

Cada libro debe tener: Nombre del Libro, Categoría, Páginas y Autor.

4. ¿Qué debe hacer la App? (CRUD):

Pantalla de Inicio: El login rosa y negro.

Pantalla de Bienvenida: Un panel con el icono del libro y un botón que diga 'GESTIONAR INVENTARIO'.

Pantalla de Inventario: Una lista/tabla elegante con todos los libros.

Debe tener un botón de Agregar (Rosa fuerte).

Cada libro debe tener un icono de Editar (naranja) y uno de Eliminar (rojo).

Confirmaciones: Si voy a borrar un libro, que salga un mensaje preguntando '¿Estás seguro de eliminar esta lectura?'.

5. Organización de los Archivos (lib/):

main.dart: Configuración de Firebase y rutas.

login_page.dart: Pantalla de entrada.

menu_page.dart: El panel de bienvenida.

inventario_page.dart: La lista de libros.

widgets/: Una carpetita para los cuadros de diálogo de 'Agregar', 'Editar' y 'Eliminar'.

Haz que el código sea limpio, que los botones brillen en rosa fuerte al tocarlos y que todo se guarde en la nube de Firebase. ¡Empecemos!"
